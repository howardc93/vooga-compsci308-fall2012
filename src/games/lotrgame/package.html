This package is part of the CompSci 308 VOOGA Game Engine Project. This package contains the Lord of the Rings game created by Michael Deng and Jei Min Yoo.
This game demonstrates the use of the platormer's game engine. In this game, we are demonstrating the game engine's ability to:
- create a bunch of levels
- set different winning/losing conditions for each level
- use our own characters and images
- use the inputs teams functionality
- make fun games quickly
